OpenStack实验环境配置

1.配置Controller node（管理节点）为时间服务器，并配置其他节点从管理节点同步时间

1.1.修改Controller node（管理节点）的配置文件。添加如下内容：

allow 10.0.0.0/24
bindcmdaddress 0.0.0.0
local stratum 10

启动chrony服务，并设置开机自启

systemctl enable chronyd.service
systemctl start chronyd.service

1.2.修改其他节点的/etc/chrony.conf配置文件。

server controller iburst

1.3.检验

chronyc sources

2.配置yum源（/etc/yum.repo.d/a.repo)

 [OpenStack-mitaka]
 name = OpenStack Mitaka Repository
 baseurl = http://mirrors.yun-idc.com/centos/7.2.1511/cloud/x86_64/openstack-mitaka/
  #http://mirrors.yun-idc.com 是国内比较快的OpenStack的yum源
 gpgcheck=0
 enabled=1


 [base]
 name = base
 baseurl = http://mirrors.yun-idc.com/centos/7.2.1511/os/x86_64/
 enabled=1
 gpgcheck=0



[extras]
name = extras
baseurl = http://mirrors.yun-idc.com/centos/7.2.1511/extras/x86_64/
enabled=1
gpgcheck=0

3.安装OpenStack客户端

yum -y install python-openstackclient
yum -y install openstack-selinux

4.安装Mariadb数据库

4.1.安装软件包

yum -y install mariadb mariadb-server python2-PyMySQL

4.2.编辑文件/etc/my.cnf

[mysqld]
default-storage-engine=innodb
innodb_file_per_table
collation-server=utf8_general_ci
init-connect='SET NAMES utf8'
character-set-server=utf8

4.3.初始化

mysql_install_db --datadir="/var/lib/mysql" --user="mysql"

4.4.完成启动

systemctl start mariadb.service

4.5.设置数据库管理员root的密码

mysql_secure_installation

5.NoSQL数据库(管理节点上安装配置MongoDB)

##5.1.安装MongoDB软件包 ##

yum -y install mongodb mongodb-server

5.2.编辑/etc/mongod.conf文件

#配置bind_ip项的值为管理节点的IP地址
 bind_ip[=*.*.*.*

5.3.完成安装启动

systemctl start mongod.service

6.消息队列服务（管理节点)(安装RabbitMQ消息队列服务)

6.1.安装软件包

yum -y install rabbitmq-server

6.2.启动

systemctl start rabbitmq-server.service

6.3.添加用户并为用户赋予读和写的访问权限

rabbitmqctl add_user 用户名 密码
rabbitmqctl set_permissions 用户名 ".*"".*"".*"

7.Identity服务认证机制（管理节点安装Memcached服务）（缓存令牌）

##7.1.安装软件包##

yum -y install memcached python-memcached

7.2.启动服务

systemctl start memcached.service

Ubuntu16.0.4搭建JSP环境和LAMP环境

2017/11/19 9:15:22

JSP环境的搭建

1. 安装JDK

（1）下载jdk

由于16.0.4版本的软件仓库中没有jdk文件，所以要从Oracle的官网上下载。

下载地址：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

这里我下载的是jdk-8u151-linux-i586.tar.gz。

（2）解压Jdk软件包并将解压的文件夹复制到/usr/lib/jvm目录下，无需安装

`sudo tar -zxvf jdk-8u151-linux-i586.tar.gz `
`sudo cp -r jdk-8u151-linux-i586 /usr/lin/jvm/ `

(3)设置环境变量

修改/etc/profile文件，在后面添加如下内容，然后输入 source /etc/profile使设置立即生效

#set java environment
　　export JAVA_HOME=/usr/lib/jvm/jdk1.8.0_25
　　export JRE_HOME=${JAVA_HOME}/jre`
　　export CLASSPATH=./:${JAVA_HOME}/lib:${JRE_HOME}/lib
　　export PATH=${JAVA_HOME}/bin:$PATH

对于root权限还要修改在/etc/environment文件,在$PATH末尾添加jdk下的bin目录路径，我的是/usr/lib/jvm/jdk-8u151-linux-i586/bin

`PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games:/usr/lib/jvm/jdk-8u151-linux-i586/bin`

输入source /etc/environment使设置立即生效

（4）验证，在终端中分别输入java javac，如果没有报错就安装成功。

2. 安装Apache2

在终端敲命令就行了：
sudo apt-get install apache2

3. 安装MySQL

sudo apt-get install mysql-client mysql-server

查看端口情况：netstat -tap

进入MySQL数据库：mysql -uroot -pkey（root用户名 key密码）

4. 安装Tomcat

下载地址：https://tomcat.apache.org/download-80.cgi

选择合适的tar.gz包下载，然后解压，把解压后的文件移到/usr/local/

tar -zxvf apache-tomcat-8.5.23.tar.gz
cp -r apache-tomcat-8.5.23 /usr/local/

启动Tomcat,在终端执行/usr/local/apache-tomcat-8.5.23/bin/startup.sh

关闭为shutdown.sh

LAMP环境的安装

注：PHP一定要最后一个安装，否则很有可能会出现配置问题。

1. 安装MySQL

同上。

2. 安装Apache2

同上。

3. 安装PHP

sudo apt-get install php7.0

测试：php -v
